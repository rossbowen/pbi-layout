# PBI Layout

Design Power BI report layouts on a 12-column grid.

Use it to plan where headers, footers, sidebars and visuals go before you build anything in Power BI. The pixel values you see match what Power BI will use.

## What you can do

- Drop elements onto a 12-column grid. They snap to the grid.
- Turn the header, footer and sidebar on or off. The layout adjusts.
- Read exact X, Y, width and height values for each element from the spec table.
- Share a layout by URL.
- Export the layout as a PBIR folder and open it in Power BI Desktop developer mode.

## Run it locally

You need Node.js 22 or later.

```
npm install
npm run dev
```

## Build

```
npm run build
```

The built files go in `dist/`.

## Test

```
npm test
```
